APCA Script Test

Usage:

First color must be text, second color must be the background.

APCAcontrast() requires Ys (display luminance)
sRGBtoY() requires an sRGB color array
displayP3toY() requires a displayP3 RGB color array
colorParsley() takes anything and parses it to an RGB color array

Lc = APCAcontrast(sRGBtoY(textColor),sRGBtoY(backgroundColor));
